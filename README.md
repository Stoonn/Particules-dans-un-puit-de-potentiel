# Particules-dans-un-puit-de-potentiel
Projet semestriel d'informatiques L2 UVSQ


## Installation

1. Cloner le repo

2. Créer et activer le venv :
```bash
   python3 -m venv venv
   source venv/bin/activate
```
3. Installer les dépendances système (tkinter) :
- **Linux** : `sudo apt install python3-tk`
- **Windows** : inclus avec Python, rien à faire
- **Mac** : `brew install python-tk`

4. Installer les dépendances Python :
```bash
   pip install -r requirements.txt
```
